## Temperature Converter

Вона дозволяє швидко конвертувати температуру зі шкали Цельсія у шкалу Фаренгейта.

# Як працює програма
Програма запитує у користувача числове значення (температуру в градусах Цельсія), проводить розрахунок за формулою:
T = (celsius * 9/5) + 32
та виводить результат у консоль.

## Приклад використання
1. Запустіть файл: `converter.py`
2. Введіть температуру в Цельсіях: 25 
   Температура у Фаренгейтах: 77.0
